TODO 4: Object.keys, values, entries

📌 Tạo Object: { ten: "Minh Tuấn", diem: 6.5, lop: "B2" }.
- Dùng Object.keys() để lấy danh sách khóa (key).
- Dùng Object.values() để lấy danh sách giá trị.
- Dùng Object.entries() để lấy cặp [key, value].
- In cả ba kết quả.
